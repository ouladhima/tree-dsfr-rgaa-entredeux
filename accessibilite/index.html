Accessibilite : non conforme

Declaration d'accessibilite

Cette declaration d'accessibilite s'applique au site Cartographie RMFP. Elle constitue la version initiale du cadrage RGAA pour le service web et devra etre completee avant publication officielle.

Etat de conformite

Le site Cartographie RMFP est non conforme avec le referentiel general d'amelioration de l'accessibilite (RGAA), version 4.1.2.

A la date du 26 mars 2026, aucun audit complet de conformite n'a encore ete formalise. La presente declaration accompagne une refonte DSFR et RGAA du service.

Resultats des tests

Aucun taux de conformite n'est publie a ce stade. Les verifications menees dans ce chantier portent sur la structure HTML, la navigation clavier, les statuts dynamiques, les contrastes et les pages reglementaires.

Etablissement de cette declaration

Cette declaration a ete etablie le 26/03/2026.

Technologies utilisees pour le service web :

  • HTML5
  • CSS avec DSFR
  • JavaScript
  • JSON statique

Contenus non accessibles

Les points suivants restent a auditer ou a completer :

  • les contenus institutionnels placeholder des pages reglementaires
  • la qualification exhaustive des contrastes et des restitutions lecteurs d'ecran
  • la verification manuelle complete de tous les criteres RGAA applicables

Contenus non soumis a l'obligation d'accessibilite

Les documents PDF relies depuis la cartographie RMFP n'entrent pas dans le perimetre de cette premiere declaration et n'ont pas ete audites dans ce lot.

Retour d'information et contact

Si vous n'arrivez pas a acceder a un contenu ou a un service, vous devez pouvoir contacter l'equipe responsable du site pour obtenir une alternative accessible ou etre oriente.

Voies de recours

Si vous constatez un defaut d'accessibilite qui vous empeche d'acceder a un contenu ou a une fonctionnalite du site, que vous le signalez et que vous ne recevez pas de reponse satisfaisante, vous pouvez saisir le Defenseur des droits.

  • contacter un delegue du Defenseur des droits proche de chez vous
  • utiliser le formulaire de contact sur le site du Defenseur des droits
  • envoyer un courrier postal au Defenseur des droits
